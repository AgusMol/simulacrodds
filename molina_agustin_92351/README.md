# Examen Final

## Backend

Iniciá el backend

#### `cd backend`

instalá las dependencias

#### `npm install`

y luego 

#### `node server`

o 
#### `npx nodemon server`

## Frontend

#### `cd frontend`

instalá las dependencias

#### `npm install`

Inicialo con:

#### `npm start`

El proyecto corre en: [http://localhost:3000](http://localhost:3000)
